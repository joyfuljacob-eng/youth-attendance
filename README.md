# 📱 학익교회 청년부 예배현황 앱

학익교회 청년부 예배와 샘모임 참여 현황 웹앱입니다.

---

## 🚀 배포 방법 (GitHub → Vercel)

### 1단계: GitHub에 올리기

1. **github.com** 접속 후 로그인
2. 우측 상단 **+** 버튼 클릭 → **New repository** 클릭
3. Repository name에 `youth-attendance` 입력
4. **Create repository** 클릭
5. 새로 만든 저장소 페이지에서 **"uploading an existing file"** 링크 클릭
6. 이 폴더 안의 **모든 파일을 드래그해서 끌어다 놓기**
7. **Commit changes** 버튼 클릭

### 2단계: Vercel로 배포하기

1. **vercel.com** 접속
2. **Sign up** 클릭 → **Continue with GitHub** 으로 가입
3. **Add New Project** 클릭
4. GitHub에서 `youth-attendance` 저장소 선택
5. **Deploy** 버튼 클릭
6. 약 1-2분 후 링크 생성! 🎉

### 3단계: 스마트폰에 설치하기

#### 안드로이드 (크롬)
1. 크롬 브라우저에서 생성된 링크 접속
2. 주소창 오른쪽 **⋮** 메뉴 탭
3. **"홈 화면에 추가"** 탭
4. **"추가"** 탭 → 완료!

#### 아이폰 (사파리)
1. **반드시 사파리** 브라우저로 링크 접속
2. 하단 **공유 버튼(□↑)** 탭
3. **"홈 화면에 추가"** 탭
4. **"추가"** 탭 → 완료!

---

## ✅ 주요 기능

- 청년 명단 등록 (이름, 성별, 전화번호, 생일, 샘 배정)
- 예배 출석 체크 (날짜별)
- 샘 모임 출석 체크 (샘별, 날짜별)
- 새가족 관리 + 4주 교육 이수 체크
- 생일 알림 (오늘 생일, 이번달 생일자)
- 장기 결석 알림 (4주/8주+)
